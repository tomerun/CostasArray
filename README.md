# Costas Array Problem

Costas配列を解くプログラム

発展の方針として、次に挙げる方向性が考えられる

* あるNでの全列挙をできるだけ高速に行う
 * シングルスレッド
 * マルチスレッド/マルチコア
* 大きなNで1つの解を見つける
